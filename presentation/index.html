About Pretio

  1. We power reward-based advertising on mobile apps and FB games
  2. API-driven ad network
  3. Cool stuff

Why this Project

  • We wanted to start making visualizations of our data
  • See patterns in geographic location
  • Show traffic changes and shifts in popularity as they happen
  • Investors like pretty graphs!

Where's the Data From?

  • Requests to an API that serves mobile ad units
  • Events are published through a Redis server
  • We tapped into the traffic of one of our publishers' apps
